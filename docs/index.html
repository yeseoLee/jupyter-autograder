1. 업로드

전체 제출 ZIP 채점

브라우저 내 처리, 서버 업로드 없음

최상위 업로드 ZIP과 학생별 내부 ZIP에 공통 비밀번호가 걸려 있으면 같은 값으로 복호화합니다. 비밀번호가 없으면 비워 두면 됩니다.

ZIP 파일을 선택하면 여기에서 진행 상태를 표시합니다.

권장 입력 구조

전체제출.zip
└─ 제출파일/
   ├─ 홍길동_기본 1반/
   │  └─ 홍길동_기본 1반 2차수.zip
   └─ 김학생_기본 1반/
      └─ 김학생_기본 1반 2차수.zip

자동 보정

  • 최상위 래퍼 폴더와 제출파일/는 자동 제거
  • 학생 폴더 안에 ZIP이 없으면 직접 포함된 .ipynb도 사용
  • 노트북이 여러 개면 가장 큰 파일을 우선 선택
